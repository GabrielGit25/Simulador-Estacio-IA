# EstaIAVa

IA que gera questões variadas de Redes e Segurança para estudantes.

## O que é

**O problema:** os simulados oficiais da Estácio sempre caem as mesmas questões. Quem já fez uma prova sabe como é — decora o gabarito e passa, ou pior, fica acreditando que conhece os conceitos pelo simples fato de repetir as questões estáticas do portal da Estácio.

**A solução:** uma ferramenta interessante de estudo já que os simulados da Estácio não possuem variações de questões. A ideia é a IA gerar sempre questões novas e aleatórias para testar o conhecimento do aluno nas matérias de Redes e Segurança.

## Como funciona

- **Modos:** Misto, Só Objetivas, Só Discursivas
- **Matérias:** Fundamentos de Redes, Segurança, ou Tudo
- **Sistema de nota:** 10 questões, dashboard com progresso, resultado final (Aprovado/Reprovado)

## Como usar

1. Acesse o site
2. Clique em "Configurar API Key"
3. Pegue uma chave gratuita em [openrouter.ai/keys](https://openrouter.ai/keys)
4. Cole a chave e salve
5. Escolha o modo e matéria, clique em "Iniciar Tarefa"

## Tecnologias

- HTML, CSS, JavaScript
- OpenRouter API (modelos de IA gratuitos)
